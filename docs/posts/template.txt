🅰️Angular  1️⃣9️⃣ 🎄Advent 
Día 1️⃣ :  Setup
✅ Asegúrate de contar con un Node.js >= 20
✅ Instala Angular CLI
npm i -g @angular/cli@latest
✅ Verifica la versión de Angular CLI
ng version
✅ Crea un proyecto minimalista Angular
ng new astro-bookings --minimal -p=lab --ssr=false --style=css --experimental-zoneless
